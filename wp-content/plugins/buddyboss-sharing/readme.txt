Adds share buttons across your site to let members easily share content to social media platforms.

== Description ==

BuddyBoss Sharing enhances your BuddyBoss Platform community by adding powerful social sharing capabilities. Let members share activities, profiles, groups, and other content to popular social media platforms with beautiful Open Graph previews. Designed for seamless integration and ease of use, it helps expand your community's reach and boost engagement through social sharing.

= Key Features =

* **Activity Sharing** - Share activities to social platforms with link previews
* **Enhanced Sharing Options** - Share to groups, friends, and messages within your community
* **Custom Messages** - Allow members to add custom messages when sharing
* **Multiple Platforms** - Support for Facebook, Twitter, LinkedIn, WhatsApp, Messenger, and more
* **Open Graph Tags** - Automatic generation of rich social media previews
* **Profile & Group Sharing** - Share member profiles and groups to social media
* **Login Page SEO** - Open Graph tags for login and registration pages
* **Link Previews** - REST API for generating beautiful link previews
* **Plugin Compatibility** - Works seamlessly with popular SEO plugins
* **Site SEO Module** - Built-in SEO optimization for shared content

= Requirements =

* **WordPress** 5.0 or higher
* **PHP** 7.4 or higher
* **BuddyBoss Platform** (Required)
* **Composer** 2.x or higher (for development)

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Plugins > Add New
3. Search for "BuddyBoss Sharing"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin ZIP file
2. Log in to your WordPress admin panel
3. Navigate to Plugins > Add New > Upload Plugin
4. Choose the downloaded ZIP file and click "Install Now"
5. After installation, click "Activate Plugin"

= Configuration =

1. Ensure BuddyBoss Platform is installed and activated
2. Navigate to BuddyBoss > Settings > Activity in WordPress admin
3. Configure your activity sharing settings
4. Choose which social platforms to enable
5. Customize sharing options for groups, friends, and messages
